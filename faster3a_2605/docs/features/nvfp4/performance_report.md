## 五、量化精度

| 指标 | 值 |
|------|-----|
| 量化权重数 | 192 |
| CosSim (FP64) | ~0.9952 |
| 量化格式 | NVFP4 (E2M1 + E4M3 FP8 block scale + FP32 tensor scale) |
| 排除项 | head.weight (FP16, RL训练稳定), ffn.value.weight (FP16, cmix内核约束), 8个低秩权重 (FP16) |

---

## 六、优化技术总结

1. **位运算 IEEE 754 解码** — 将 `__constant__` 内存 LUT 查找替换为纯位运算 (`e2m1_decode_f()` / `e2m1_decode_h()`)，消除 warp 内地址发散导致的序列化瓶颈，内核带宽 80 → 350 GB/s
2. **blk16 GEMV 内核** — 单 warp 设计 (`__launch_bounds__(32, 1)`)，`uint2` 向量化 8 字节加载，M=1/M=2 decode 路径专用
3. **融合反量化-矩阵乘** — K 循环内即时反量化累加，无中间 FP16 缓冲区
4. **全内核覆盖** — 9 个 NVFP4 内核全部使用位运算解码，无任何 `__constant__` LUT 残留
